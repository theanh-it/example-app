# Bun + Typescript + vite + vue3 + pinia + capacitor

```
Bun 1.2.5
vue3
capacitor 7
```

##

```
bun run add android
bun run add ios
```

## Development

```
bun run dev
change variable dev in capacitor.config.ts "false" => "true"
bun run copy

bun run open android
bun run open ios
```

### Production

```
bun run build
change variable dev in capacitor.config.ts "true" => "false"
bun run sync
```
